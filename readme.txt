Activate asynchronous Google Analytics tracking code on your site. Just add your ID and the code appears on the right spot.

== Description ==

Adding Google Analytics to your website never has been easier. Just add your analytics ID and the correct asynchronous Javascript code is added to every page. No fiddling around anymore with manually pasting the code in your template.


== Installation ==

1. Upload `wpx-google-analytics.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Settings` > `WPX Google Analytics` and paste in your analytics ID
